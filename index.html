Política de Privacidade

Extensão: Extrator de Seguidores PRO

Última atualização: 2026-01-17

Esta Política de Privacidade descreve como a extensão Extrator de Seguidores PRO lida com dados durante o uso. A extensão foi desenvolvida para respeitar a privacidade dos usuários e seguir as políticas da Chrome Web Store.

1) Coleta e uso de dados

A extensão acessa apenas informações públicas visíveis nas páginas visitadas pelo próprio usuário, como nomes de usuário e imagens de perfil públicas, com o objetivo de organizar e exportar esses dados a pedido do usuário.

  • Não coletamos dados pessoais sensíveis (ex.: documentos, senhas, dados financeiros).
  • Não coletamos conteúdo privado protegido por login além do que o usuário vê na página aberta.

2) Processamento local

Os dados são processados localmente no navegador. A extensão não envia informações para servidores externos próprios.

3) Armazenamento

A extensão pode armazenar preferências do usuário e dados temporários localmente no navegador (por exemplo, para salvar progresso e facilitar a exportação).

4) Downloads e exportação

O usuário pode exportar os dados extraídos para arquivos locais (por exemplo, JSON, CSV ou TXT). Esses arquivos são salvos no dispositivo do usuário.

5) Compartilhamento de dados

A extensão não vende, não compartilha e não transfere dados do usuário para terceiros.

6) Código remoto e publicidade

A extensão não executa código remoto. Todo o código é empacotado e distribuído por meio da Chrome Web Store. Podem existir links de publicidade que redirecionam o usuário para sites externos, sem coleta de dados pela extensão.

7) Segurança

Buscamos minimizar o acesso a dados e manter o funcionamento restrito às permissões necessárias para a finalidade da extensão.
